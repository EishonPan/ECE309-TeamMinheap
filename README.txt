The connect.cpp program is a connect 4 game that can be played with 2 real players, 1 player vs computer, or 2 computer players playing.
No significant changes have been made to the rules of a classic connect 4 game.

Compile on mac by compiling with make connect, then running the program with ./connect

This program was made by Will Harger, Charles Terry, and Yixuan Pan

Interesting features include a printed display of the game board updated every turn and a devloped Ai computer player that will attempt to break and chains of player pieces.
